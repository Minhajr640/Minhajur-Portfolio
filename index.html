    Projects

    Spring Social Media Blog

    For this project, I developed the controller, service, and data repository layers for the backend API of a Social Media Blog using Spring Boot. Each layer was configured using Spring's stereotype annotations. For the data repository layer, I implemented JPARepository interface, which provides easy access to database interactions while reducing boilerplate code. I also developed service layers for business logic and to abstract the database access implementations provided by the repository class. Finally, I created a controller layer annotated with @RestController and used Spring Boots request mapping methods to map requests to their respective methods and returned the responses using HttpEntity class. Throughout the application, constructor injections was used to provide dependencies for each class.

    Rest Social Media Blog

    For this project, I developed the controller, service, and data access layers for the backend API for a Social Media Blog. In the data access layer, I used JDBC to handle data persistence, making sure data was being stored in the database in a consistent manner and retrieved accurately for user requests. I also implemented service layers to implement the business logic for each type of request where filtering and input validation was implemented. Finally, in the controller layer, I utilized Javalins Context object along with Jackson's Object Mapper class to handle incoming requests and send back appropriate responses. Using this layered architecture, I developed a backend API that abstracts data access layer from users, processes their requests, and ensures data is handled effectively.
